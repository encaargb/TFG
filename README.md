# Annot8

Annot8 is a web-based document annotation tool developed as a university final degree project. The actual application is located in the `annot8-web-annotation-tool/` folder.

## Repository Structure

```text
.
  annot8-web-annotation-tool/   Main Annot8 application
    frontend/                   Vue frontend application
    backend-mock/               Mock backend for development/demo persistence
```

## Getting Started

Enter the project folder before running setup, development or test commands:

```bash
cd annot8-web-annotation-tool
```

For full setup instructions, technical details and project documentation, see:

```text
annot8-web-annotation-tool/README.md
```
